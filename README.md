PWB (Half-Life: 1 weapons branch)
---------------------------------

PWB is a brand new weapons base, built to be a simple as possible (without replacing the entire Teardown API with "simpler" functions to allow for easy modification.

PWB has support for any weapon type (as long as you feel like coding it) and extremely basic weapon animations (but can support more complex math based animations [see earlier "()" message])

Lackluster instructions on how to set up your own weapons can be found at the top of main.lua, Teardown API documentation can be found at https://teardowngame.com/experimental/api.html
